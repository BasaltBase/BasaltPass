# 测试账户信息

## 默认测试账户

由于后端服务需要先启动并创建用户，这里提供一些测试用的账户信息：

### 方法1：通过前端注册页面
1. 访问 http://localhost:5173/
2. 点击"注册"按钮
3. 填写以下信息：
   - 邮箱：test@example.com
   - 密码：password123
   - 确认密码：password123

### 方法2：使用预设测试账户
如果后端已经创建了测试用户，可以使用：
- **邮箱**：test@example.com
- **密码**：password123

### 方法3：OAuth登录
- 点击"使用Google登录"
- 点击"使用Microsoft登录"
- 点击"使用Meta登录"

## 功能测试

登录后可以测试以下功能：
1. **个人资料** - 修改头像、昵称
2. **钱包管理** - 查看余额、充值、提现
3. **安全设置** - 2FA双因素认证
4. **管理后台** - 用户管理、审计日志

## 注意事项

- 确保后端服务运行在 http://localhost:8080
- 确保前端服务运行在 http://localhost:5173
- 首次使用需要先注册账户
