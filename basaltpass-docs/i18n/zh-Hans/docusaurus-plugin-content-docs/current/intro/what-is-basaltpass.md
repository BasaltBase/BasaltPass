# 什么是 BasaltPass？

BasaltPass 是一个现代、安全且支持多租户的身份验证系统，专为与第三方应用程序无缝集成而设计。它提供了一个强大的身份提供商（IdP）解决方案，支持 OAuth 2.0 和 OpenID Connect (OIDC) 等标准协议。

## 核心特性

-   **多租户架构**: 内置对多租户的支持，实现严格的数据隔离。
-   **标准协议**: 完全符合 OAuth 2.0 和 OIDC 标准，具有通用兼容性。
-   **安全性**: 支持 PKCE (Proof Key for Code Exchange) 以实现安全的公共客户端身份验证。
-   **RBAC**: 细粒度的基于角色的访问控制，用于管理用户权限。
-   **开发者友好**: 提供全面的 API 和清晰的文档，易于集成。

## 适用对象

-   **开发者**: 以最小的精力将安全身份验证集成到您的应用中。
-   **企业**: 在单个部署实例中管理多个组织（租户）。
-   **SaaS 提供商**: 为您的客户提供身份验证即服务。
