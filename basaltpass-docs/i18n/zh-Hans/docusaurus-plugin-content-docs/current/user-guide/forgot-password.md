# 忘记密码

自助密码重置流程允许用户在无需管理员干预的情况下重新获得访问权限。

## 流程

1.  用户在登录页面点击 **忘记密码**。
2.  用户输入其注册的电子邮件地址。
3.  BasaltPass 发送一封包含唯一、限时链接的电子邮件。
4.  用户点击链接并输入新密码。

## 前提条件

-   **邮件服务**: 必须已配置 (SMTP/SES 等)。
-   **已验证的邮箱**: 用户必须有已验证的电子邮件地址 (可选，取决于设置)。

## 安全性

-   链接在短时间后过期 (例如 15 分钟)。
-   链接使用后即失效。
