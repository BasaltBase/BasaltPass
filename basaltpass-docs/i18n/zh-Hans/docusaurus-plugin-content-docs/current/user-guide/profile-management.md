# 个人资料管理

用户可以通过 **用户控制台** 管理其个人信息。

## 可编辑字段

-   **头像**: 上传个人头像。
-   **昵称**: 显示名称。
-   **手机号**: 联系电话 (可能需要验证)。
-   **密码**: 修改当前密码。

## 只读字段

-   **邮箱**: 注册后通常锁定，或需要特殊流程更改。
-   **角色**: 仅由租户管理员管理。
-   **租户 ID**: 固定不变。

## 关联账户

用户可以查看和解绑与其个人资料关联的第三方社交账户 (Google、GitHub)。
