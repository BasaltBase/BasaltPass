# What is BasaltPass?

BasaltPass is a modern, secure, and multi-tenant authentication system designed for seamless integration with third-party applications. It provides a robust Identity Provider (IdP) solution supporting standard protocols like OAuth 2.0 and OpenID Connect (OIDC).

## Core Features

-   **Multi-Tenancy**: Built-in support for multiple tenants with strict data isolation.
-   **Standard Protocols**: Full compliance with OAuth 2.0 and OIDC for universal compatibility.
-   **Security**: Supports PKCE (Proof Key for Code Exchange) for secure public client authentication.
-   **RBAC**: Granular Role-Based Access Control for managing user permissions.
-   **Developer Friendly**: Comprehensive APIs and clear documentation for easy integration.

## Who is it for?

-   **Developers**: Integrate secure authentication into your apps with minimal effort.
-   **Enterprises**: Manage multiple organizations (tenants) within a single deployed instance.
-   **SaaS Providers**: Offer authentication as a service to your customers.
