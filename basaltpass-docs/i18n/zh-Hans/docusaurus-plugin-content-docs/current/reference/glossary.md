# 术语表

BasaltPass 文档中常用术语的定义。

-   **访问令牌 (Access Token)**: 用于访问受保护资源的短期凭证。
-   **授权码 (Authorization Code)**: 用于交换访问令牌的临时代码。
-   **客户端 (Client)**: 尝试访问用户账户的应用程序。
-   **授权类型 (Grant Type)**: 客户端获取访问令牌所使用的方法。
-   **身份提供商 (IdP, Identity Provider)**: 认证用户的服务 (即 BasaltPass)。
-   **JWT (JSON Web Token)**: 一种紧凑的、URL 安全的声明表示方式，用于在两方之间传输。
-   **OIDC (OpenID Connect)**: 基于 OAuth 2.0 协议之上的身份层。
-   **资源服务器 (Resource Server)**: 托管用户数据的 API。
-   **租户 (Tenant)**: BasaltPass 实例中的隔离环境。
